By using the associativity axiom, verify the following where
$a, b, c, d$ are elements of a semigroup.
\begin{myenum}
\item $(a * b) * (c * d) = a * (b * (c * d))$
\item $a * ((b * c) * d) = (a * (b * c)) * d$
\item $((a * b) * c) * d = (a * b) * (c * d)$
\end{myenum}
